## 学到了什么
- argparse 是 Python 标准库的命令行参数解析器，以后所有 CLI 工具都会用到
- 位置参数 vs 可选参数：source 是必填的，--dry-run 是可选的
- action="store_true"：带上 --dry-run 就是 True，不带就是 False
- --dry-run 预览模式是专业 CLI 工具的常见设计，先看再动手，防止误操作
- git config --global 设置作者信息，noreply 邮箱保护隐私
