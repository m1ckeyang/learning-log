# AI 应用工程师转型计划

## 这个文件夹里有什么

| 文件 | 说明 |
|------|------|
| `ai-engineer-roadmap.html` | 6个月转型路线图（用浏览器打开） |
| `学习日志模板.md` | 每日学习日志模板（每天复制一份使用） |
| `周复盘模板.md` | 每周复盘模板（每周日填写） |

## 怎么开始

1. **今天**：用浏览器打开 `ai-engineer-roadmap.html`，通读一遍
2. **明天**：注册 GitHub 账号，创建 `learning-log` 仓库
3. **后天**：开始阶段 0 的环境搭建

## 每日流程

1. 开工前：复制 `学习日志模板.md`，重命名为今天的日期
2. 写下今天要完成的一件小事
3. 写代码（3 小时，用番茄钟）
4. 结束前：填写日志，Git 提交

## 每周流程

1. 周一到周六：按计划学习和写代码
2. 周日：填写 `周复盘模板.md`，整理代码，看本周 Git 提交记录

## 遇到问题怎么办

- 代码报错 → 把报错信息贴给 AI 助手
- 不知道用什么方案 → 问 AI 助手
- 卡了超过 2 天 → 重新缩小项目范围，先完成再完美

- 第 14 周末：AI 对话助手能用了
